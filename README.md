**Project by:** NTI Team

## Overview
Game Store API is a backend system for managing games, user authentication, favorites, cart & orders, and community interactions.  
It supports a gaming platform where users can browse games, manage favorites, place orders, and interact in real-time.

---

## Features

### 🔴 Must Have
- User authentication & setup
- Games API (CRUD operations)
- Favorites management
- Cart and order processing
- Community features & real-time messaging (Socket.io)

### 🟡 Should Have
- Game rating & review system
- Enhanced order tracking
- User profile management

### 🟢 Could Have
- Game recommendations
- Advanced search & filtering
- Gamification elements (badges, achievements)
